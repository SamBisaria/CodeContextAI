## Getting Started 

### Prerequisites

- Node.js version >= 18.13.0
- Google gemini pro api key


### Clonning

```bash
git clone https://github.com/rishabhknowss/QuineLaw.git
```

### Installation

```bash 
 npm install
 npm run dev
 ```
